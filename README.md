# Progetto Manutenzione Preventiva — Esacottero F550

> Corso di Manutenzione Preventiva — A.A. 2025/2026

## Descrizione

Progetto universitario incentrato sulla manutenzione preventiva di un esacottero basato su frame DJI F550, equipaggiato con flight controller Picshawk 6X e firmware PX4.

## Struttura della repository

```
manutenzione-preventiva-freddi/
├── README.md                  # Questo file
├── docs/
│   ├── BOM.md                 # Bill of Materials completa
│   └── datasheets/            # Datasheet dei componenti (PDF)
├── img/                       # Immagini e schemi del drone
└── maintenance/
    └── (piani di manutenzione, checklist, analisi FMEA)
```

## Componenti principali

- **Frame**: DJI F550 (esacottero)
- **FC**: Picshawk 6X
- **Motori**: AIR2213 KV920 (x6)
- **ESC**: Tekko32 F4 (x6)
- **GPS**: CubePilot Here+ con base RTK
- **Telemetria**: Holybro 433 MHz
- **Batteria**: LiPo 4S 5600 mAh
- **Firmware**: PX4 + QGroundControl
